DeltaForce API
===============

## 特性

* 基于PHP`8.0+`

> DeltaForce API的运行环境要求PHP8.0+

## 文档

[完全开发手册](https://doc.thinkphp.cn)


## 安装

~~~
composer create-project coolxitech/deltaforce deltaforce
~~~

启动服务

~~~
cd deltaforce
php think run
~~~

然后就可以在浏览器中访问

~~~
http://localhost:8000
~~~

如果需要更新框架使用
~~~
composer update coolxitech/deltaforce
~~~
